# Do-it-frontend

## Petite notice pour lancer le projet.

## Installation

Apres avoir cloné le projet, naviguez vers le repertoire du projet: cd do-it-frontend

Installer les dépendances avec: npm install

Creer votre propre fichier ".env" basé sur le contenu de "example.env" et remplacez les valeurs par les votre.

## Lancement

Lancez d'abord le projet backend, puis lancez le frontend avec: npm run dev
